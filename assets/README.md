# Assets Folder

This folder should contain your app assets. For now, Expo will use default assets.

## Required Assets (Optional - Expo will use defaults if missing):

1. **icon.png** - 1024x1024 PNG app icon
2. **splash.png** - 1284x2778 PNG splash screen (optional - using solid color)
3. **adaptive-icon.png** - 1024x1024 PNG Android adaptive icon (optional)
4. **favicon.png** - 48x48 PNG web favicon (optional)

## Quick Fix:

The app.json has been updated to work without these files. Expo will use default icons.

To add custom assets later:
1. Create 1024x1024 PNG icon and save as `icon.png`
2. Create splash screen and save as `splash.png`
3. Place all files in this `assets/` folder

For now, the app will run without custom assets!
